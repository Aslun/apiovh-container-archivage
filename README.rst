Debian & Distributions dérivées de Debian
----------

Installation de Python

    apt install python & python-pip

Installation pip de OVH

    pip install ovh
    
Exécution du script

    python contairerarchivage.py
    
    
CentOS and RHEL
----------

Installation de Python

    yum install epel-release & python-pip

Installation pip de OVH

    pip install ovh
    
Exécution du script

    python contairerarchivage.py
